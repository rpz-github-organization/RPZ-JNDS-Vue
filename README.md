# RPZ-JNDS-Vue

## 项目启动 Project setup
```
# 依赖安装 Dependencies installation
yarn install

# 开发编译与热重载部署 Compiles and hot-reloads for development
yarn run serve


# 编译构建与压缩到生产级别 Compiles and minifies for production
yarn run build


# 运行测试 Run your tests
yarn run test


# 代码格式化 Lints and fixes files
yarn run lint (--fix)
```
